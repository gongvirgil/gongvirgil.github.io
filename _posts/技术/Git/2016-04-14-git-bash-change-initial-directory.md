每次打开`Git Bash`时，默认的初始目录是`/c/Windows/system`,切换工作目录还需要手动敲`cd /../..`，太过麻烦。

右键点击`Git Bash`启动项，打开属性，找到快捷方式的起始位置一栏，改为自己的工作目录路径就可以了。
