### 查看状态

```bash
$ git status
```

### 添加修改

```bash
$ git add .
```

### 提交修改

```bash
$ git commit -m "..."
```
### 自动追踪文件，包括你已经手动删除的，状态为Deleted的文件

```bash
$ git add -u
$ git rm .../... -r -f
$ git checkout .
$ git commit -m "..."
$ git push origin master
```

### 发布文章

```bash
rake post title="文章标题"
rake page name="页面名称.md"
rake page name="pages/页面名称.md"
```
